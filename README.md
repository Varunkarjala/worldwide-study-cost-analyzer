# worldwide-study-cost-analyzer
A Python-based data analysis project that compares global education costs including tuition, accommodation, and living expenses across 50+ countries using Pandas, NumPy, Matplotlib, and Seaborn.
This project analyzes and visualizes the cost of studying worldwide using real datasets from UNESCO, World Bank, QS Rankings, and Numbeo. It covers data cleaning, currency normalization, clustering, and interactive visualizations to help students make informed decisions about studying abroad
